Strings, Arrays, and Objects

Make working with strings, arrays, and objects easier and more productive.

Learn the ins-and-outs of strings, arrays, and objects—and all that they can do—in about an hour.

This guide is short, focused, and made for beginners. Get hands-on learning and grow your portfolio with a real project.

What you’ll learn

  • How to remove whitespace from a string.
  • How to transform text to uppercase and lowercase.
  • How to convert strings to integers.
  • How to replace a portion of text with different text.
  • How to get a portion of a string.
  • How to split a string into an array based on a character.
  • How to add items to an array or object.
  • How to merge two or more arrays or objects together.
  • How to put it all together and write a working project with string, array, and object manipulation.
